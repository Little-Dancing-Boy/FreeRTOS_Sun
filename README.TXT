◆实验器材:
	战舰STM32F103开发板
	
◆实验目的:
	学习GPIO作为输出的使用
	
◆硬件资源:
	1,DS0(连接在PB5) 
	2,DS1(连接在PE5)
	
◆实验现象:
	本实验通过代码控制开发板上的两个LED：DS0和DS1交替闪烁，实现类似跑马灯的效果。
